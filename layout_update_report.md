# StudyRoute 상세 페이지 레이아웃 수정 보고서

작성일: 2026-07-02

## 작업 범위

- 상세 페이지 UI/UX 레이아웃을 `H1 → 공통 이미지 → 본문 → 관련 링크` 순서로 정리했습니다.
- 본문 문장, title, slug, canonical, breadcrumb, JSON-LD, robots, sitemap, URL, 내부 링크 대상은 변경하지 않았습니다.
- 링크 영역의 표시 제목만 사용자 친화적인 명칭으로 변경했습니다.

## 수정 파일

- `templates/pages/content.html`
- `assets/css/main.css`
- `generator.py`

## 레이아웃 변경

- 기존 2컬럼 상세 레이아웃을 단일 읽기 흐름으로 변경했습니다.
- H1은 항상 상세 페이지 최상단에 표시되도록 통일했습니다.
- 공통 이미지는 H1 바로 아래에 가로 100% 영역으로 배치했습니다.
- 본문은 이미지 아래에서 시작하도록 정리했습니다.
- 관련 링크 섹션은 본문 뒤에 표시되도록 템플릿 안으로 이동했습니다.

## 본문 디자인 개선

- 본문 최대 폭을 제한하여 긴 줄을 줄였습니다.
- 줄 간격과 문단 간격을 넓혀 가독성을 개선했습니다.
- H2, H3, 목록, 인용문, 표 스타일을 추가했습니다.
- 모바일에서 문단 줄바꿈이 자연스럽게 되도록 보정했습니다.

## 링크 영역 명칭 변경

- `하위 지역` → `인근 지역`
- `같은 지역의 학습 허브` → `같은 지역의 학습 정보`
- `같은 지역의 형제 허브` → `같은 지역의 학습 정보`
- `하위 지역의 같은 허브` → `주변 지역의 학습 정보`
- `지역별 ...` → `관련 지역`

## 링크 디자인 개선

- 단순 버튼형 링크를 카드/태그형 링크로 개선했습니다.
- 링크 간격과 최소 높이를 확보했습니다.
- Hover 시 테두리, 그림자, 이동 효과를 추가했습니다.
- 모바일에서는 2열 또는 자동 줄바꿈 흐름을 유지합니다.
