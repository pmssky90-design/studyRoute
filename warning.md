# Warning / Low / 운영 참고 사항

최종 자동 감사 기준 Low 이슈는 없습니다.

운영 참고 사항:

- 사이트는 한국어 단일 언어 사이트이므로 hreflang은 필수 항목이 아닙니다.
- 상세 페이지 공통 이미지는 WebP로 최적화했지만 원본 구성이 매우 긴 세로형입니다. 사용성 데이터에 따라 향후 여러 섹션 이미지로 나누면 체감 성능을 더 개선할 수 있습니다.
- 현재 검색 아이콘은 UI만 존재합니다. 검색 기능을 추가할 때 `aria-label`, 키보드 포커스, 자동완성 접근성을 함께 구현하는 것이 좋습니다.
- Cloudflare Pages 또는 GitHub Pages 배포 후 실제 도메인에서 canonical, sitemap, robots 접근을 한 번 더 확인하는 것을 권장합니다.
