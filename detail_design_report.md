# StudyRoute 상세 페이지 디자인 개선 보고서

작성일: 2026-07-02

## 작업 범위

- 상세 페이지 UI/UX 디자인만 개선했습니다.
- 본문 내용, title, slug, URL, breadcrumb, canonical, JSON-LD, robots, sitemap, 내부 링크 대상은 변경하지 않았습니다.
- 이번 작업에서 `generator.py`는 수정하지 않았습니다.

## 수정 파일

- `assets/css/main.css`

## 공통 이미지 개선

- 기존 이미지 잘림 원인이던 고정 `aspect-ratio`와 `object-fit: cover`를 제거했습니다.
- 공통 이미지는 다음 방식으로 표시되도록 수정했습니다.
  - `width: 100%`
  - `height: auto`
  - `object-fit: contain`
  - 원본 비율 유지
- 원본 이미지 크기: `800 x 8000`
- 이미지 전체가 페이지 안에서 잘리지 않고 보이도록 확인했습니다.

## 본문 레이아웃 개선

- 상세 페이지 본문 최대 폭을 약 `820px` 중심으로 제한했습니다.
- 전체 상세 컨테이너는 `860px` 중심으로 정리했습니다.
- 본문 위에 얇은 구분선을 추가해 이미지와 본문 시작점이 자연스럽게 분리되도록 했습니다.
- 본문 글자 크기와 줄간격을 조정했습니다.
  - 기본 본문: `16px`, `line-height: 1.9`
  - 데스크톱 본문: `17px`
- 문단 간격과 여백을 넓혀 정보 사이트형 읽기 흐름을 개선했습니다.

## 본문 요소 스타일

- H2/H3 제목 크기와 위아래 여백을 보강했습니다.
- 목록 `ul`, `ol`, `li` 간격을 정리했습니다.
- `blockquote`는 녹색 포인트 라인과 연한 배경을 적용했습니다.
- `table`, `th`, `td`는 경계선과 헤더 배경을 적용했습니다.
- `details`, `.faq`, `.faq-item`, `summary` 스타일을 추가해 FAQ 구조도 자연스럽게 표시되도록 했습니다.

## 관련 링크 디자인

- 기존 단순 버튼형 링크를 카드/태그형 링크로 개선했습니다.
- 링크 최소 높이를 키워 클릭 영역을 확보했습니다.
- Hover 시 다음 효과를 적용했습니다.
  - 테두리 색상 변경
  - 녹색 강조색 적용
  - 부드러운 그림자
  - 살짝 떠오르는 이동 효과
- 데스크톱에서는 3열, 태블릿은 3열, 모바일은 2열 흐름을 유지합니다.
